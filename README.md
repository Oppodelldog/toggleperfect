# toggle perfect

Hobby project to control a 3d printed device.
Find the model on (www.thingiverse.com)[https://www.thingiverse.com/thing:4295022]

## Platform
* Raspberry Pi Zero (linxy arm)

## Controlled devices
* waveshare ink (2.7")
* 4 display input keys
* 4 LEDs
